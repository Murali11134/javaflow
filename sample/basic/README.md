# JavaFlow Basic Sample

This folder contains a tiny Java project that can be used to test JavaFlow quickly.

## Classes

- `User` — simple domain model.
- `UserRepository` — in-memory repository.
- `UserService` — service that creates and loads users.

## How to test with JavaFlow

1. Open this repository in VS Code.
2. Right-click `sample/basic/src/main/java/com/example`.
3. Select `JavaFlow: Show Mindmap for Folder`.
4. Use search for `UserService`, `register`, `save`, or `findById`.

This sample is intentionally small so new users can immediately understand what JavaFlow extracts and visualizes.
